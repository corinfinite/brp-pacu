BRP-PACU
Instruction for Macintosh
===  FOR USERS  ===

Dependencies:

To build the system, you should first download and install:

Jack-OSX from http://www.jackosx.com/

The GTK-OSX Framework from http://www.gtk-osx.org/

To run the program, download the binary distribution image, open it, and double-click on the PRB-PACU icon. You can drag the application anywhere on your system, preferably into the Application directory. 

=== FOR DEVELOPERS ===

If you want to modify the program, you have to use the source distribution. You will find it at:
 
https://sourceforge.net/project/showfiles.php?group_id=204657&package_id=304598

Building instruction for Mac 

1. Dependencies:

To build the system, you should first download and install:

Jack-OSX from http://www.jackosx.com/

The GTK-OSX Framework from http://www.gtk-osx.org/

Also, you will need the development versions of:

fftw version 2.1.5 from http://www.fftw.org/download.html
(fttw version 3 does not work yet)
libglade from ftp://ftp.gnome.org/mirror/gnome.org/sources/libglade/
GtkDatabox from https://sourceforge.net/projects/gtkdatabox/

Before installing these packages, do the following from the command line:

export PKG_CONFIG_PATH='/usr/lib/pkgconfig/:/usr/local/lib/pkgconfig/:/Library/Frameworks/Gtk.framework/Resources/dev/lib/pkgconfig:/Library/Frameworks/GLib.framework/Resources/dev/lib/pkgconfig:/Library/Frameworks/Cairo.framework/Resources/dev/lib/pkgconfig:/Users/andreas/Desktop/develop/gtkdatabox-0.9.0.1/'
sudo cat > /usr/share/aclocal/dirlist << xxx
/usr/local/share/aclocal
xxx

then follow the usual installation procedure (see INSTALL)

Use the same procedure for installing BRP-PACU from the source distribution.

To make the disk image containing the BRP-PACU application, enter:

make macdist
